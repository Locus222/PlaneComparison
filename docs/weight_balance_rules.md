nemenne podminky:

PAYLOAD je jedine soucet pilota a pasazeru a jejich nakladu. toto  je definice a zakladni hodnota.
Trip fuel je palivo nutne k provedeni letu z A do B
reserve fuel je palivo pro pistove motory nutne na 30minut letu, nebo na dobu definovanou na karte obecne

Fuel on board = trip fuel + fuel resserve

useful load >= trip load

trip load = FOB + payload
 
podminka: useful load >= trip load z toho plyne MTOW >= TOW

load reserve =  useful load - trip load
